# Changelog

## [0.1.1] - 2026-07-04

### 改善
- アプリ内アップデート通知を有効化（起動時に GitHub Releases の新バージョンを確認し、バナーで通知）

## [0.1.0] - 2026-05-30

### 初回リリース

#### 主な機能
- Web記事の登録・管理（手動 / クリップボード自動検出）
- 日本語記事: VOICEVOX による高品質読み上げ
- 英語記事: Piper TTS (en_US-ryan-high) による英語読み上げ
- 言語自動検出（日本語 / 英語）
- バックグラウンド音声合成・キャッシュ
- 再生キュー管理（ドラッグ&ドロップ並び替え）
- テキストビューア（ハイライト追従・自動スクロール）
- ワードクラウド（TF-IDF / 英語 Porter2 語幹処理）
- カレンダーヒートマップ付き履歴・統計
- ライト / ダークテーマ
- キーバインドカスタマイズ
- 初回セットアップ（VOICEVOX 自動ダウンロード）
- Piper TTS セットアップ（設定タブからインストール）
- ログインサイト対応（Cookie 共有 WebView2）
